Introduction to HTML

Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages and web applications. HTML structures the content on the web, using elements that define the different parts of a webpage, such as headings, paragraphs, links, and images.

Breaking down the name:

  • HyperText: Refers to the links between web pages, enabling navigation across the internet.
  • Markup: Refers to the way HTML tags structure and define content.
  • Language: Indicates that HTML has a specific syntax or set of rules to follow.

Importance of HTML:

HTML is essential because it:

  • Provides the structure for all websites.
  • Is easy to learn and use.
  • Works in combination with CSS (for styling) and JavaScript (for functionality) to create dynamic web pages.

Basic Structure of an HTML Document:

Below is an example of a simple HTML document:

  <!DOCTYPE html>
  <html>
      <head>
          <title>My First HTML Page</title>
      </head>
      <body>
          <h1>Welcome to HTML!</h1>
          <p>HTML is easy and fun to learn.</p>
      </body>
  </html>

Explanation:

  • <!DOCTYPE html>: Declares the document type as HTML5.
  • <html>: The root element of the document.
  • <head>: Contains metadata about the document, such as its title.
  • <title>: Specifies the title that appears on the browser tab.
  • <body>: Contains the visible content of the webpage.
  • <h1>: A heading element, used for the main title.
  • <p>: A paragraph element for textual content.

Basic HTML tags

HTML tags are the building blocks of web pages. Tags are used to define the structure and content of a webpage. They are enclosed within angle brackets (e.g., <tag>). Most HTML tags come in pairs, consisting of an opening tag and a closing tag (e.g., <p>...</p>).

Essential HTML Tags:

  • <html>: Represents the root of an HTML document. It contains all the content of the webpage.
  • <head>: Contains metadata about the document, such as its title, character set, and linked resources like stylesheets.
  • <title>: Specifies the title of the document, displayed on the browser tab.
  • <body>: Defines the main content area of the document that is visible to users.
  • <h1> to <h6>: Represent headings, with <h1> being the largest and <h6> the smallest.
  • <p>: Defines a paragraph of text.
  • <a>: Creates hyperlinks to navigate between web pages.
  • <img>: Embeds images into a webpage. It is a self-closing tag.
  • <ul> and <ol>: Represent unordered and ordered lists, respectively.
  • <li>: Defines a list item inside <ul> or <ol>.
  • <div>: Defines a division or section in a document, often used for grouping elements.
  • <span>: Used for styling inline elements without creating a new block.

Example of Basic HTML Tags:

<!DOCTYPE html>
<html>
  <head>
      <title>My First HTML Page</title>
  </head>
  <body>
      <h1>Welcome to My Website</h1>
      <p>This is a paragraph of text.</p>
      <a href="https://www.example.com">Visit Example.com</a>
      <img src="image.jpg" alt="Sample Image">
      <ul>
          <li>Item 1</li>
          <li>Item 2</li>
      </ul>
  </body>
</html>

Explanation:

  • <h1>: Adds a main heading to the webpage.
  • <p>: Adds a paragraph with descriptive text.
  • <a>: Creates a clickable hyperlink.
  • <img>: Embeds an image.
  • <ul> and <li>: Create a list of items.

HTML Text Formatting

HTML Tutorial: Text Formatting

Basic text formatting tags in HTML. These tags help you style your text in various ways.

1. Bold Text

To make your text bold, use the <b> or <strong> tag.

Example: <b>This is bold text</b>

2. Italic Text

To italicize your text, use the <i> or <em> tag.

Example: <i>This is italic text</i>

3. Underlined Text

To underline your text, use the <u> tag.

Example: <u>This is underlined text</u>

4. Strikethrough Text

To add a strikethrough effect, use the <s> tag.

Example: <s>This is strikethrough text</s>

5. Subscript Text

To write subscript text, use the <sub> tag.

Example: <sub>H<sub>2</sub>O</sub>

6. Superscript Text

To write superscript text, use the <sup> tag.

Example: <sup>x<sup>2</sup></sup>

7. Monospace Text

To display text in a monospace font, use the <code> tag.

Example: <code>This is code text</code>

8. Highlighted Text

To highlight text, use the <mark> tag.

Example: <mark>This is highlighted text</mark>

9. Inserted Text

To indicate inserted text, use the <ins> tag.

Example: <ins>This is inserted text</ins>

10. Deleted Text

To indicate deleted text, use the <del> tag.

Example: <del>This is deleted text</del>

11. Quoted Text

To display short inline quotations, use the <q> tag.

Example: <q>This is quoted text</q>

12. Blockquote

To display longer quotations, use the <blockquote> tag.

Example: <blockquote>This is a blockquote</blockquote>

Links, Images, and Buttons

Links, Images, and Buttons

Links are created using the <a> tag. Here's the basic syntax:

<a href="URL">Link Text</a>

For example, to create a link to Google:

<a href="https://www.google.com">Visit Google</a>

Images are added using the <img> tag. Here's the basic syntax:

<img src="IMAGE_URL" alt="Description" />

For example, to display an image of a cat:

<img src="cat.jpg" alt="A cute cat" />

Buttons are created using the <button> tag. Buttons are used for actions like submitting forms or triggering JavaScript functions. Here's the basic syntax:

<button>Button Text</button>

For example, to create a button labeled "Click Me!":

<button>Click Me!</button>

You can also add attributes like onclick to make the button interactive. For example:

<button onclick="alert('Hello!')">Say Hello</button>

Now you know how to add links, images, and buttons to your HTML page. Try experimenting with these elements to make your web pages more interactive and visually appealing!

Lists

HTML Lists

HTML provides two main types of lists:

  • Ordered Lists (<ol>)
  • Unordered Lists (<ul>)

Ordered Lists

An ordered list is used when the order of items is important. Here's an example: 

              <ol>
                  <li>First item</li>
                  <li>Second item</li>
                  <li>Third item</li>
              </ol>

Unordered Lists

An unordered list is used when the order of items is not important. Here's an example: 

              <ul>
                  <li>Item one</li>
                  <li>Item two</li>
                  <li>Item three</li>
              </ul>

Lists can also be nested for creating sub-lists: 

              <ul>
                  <li>Main item</li>
                  <ul>
                      <li>Sub-item 1</li>
                      <li>Sub-item 2</li>
                  </ul>
              </ul>

Table

HTML Table Tutorial

Tables are used to organize data in rows and columns. Here's an example of how to create a basic table in HTML.

HTML Code

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
        <th>Header 3</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
        <td>Data 3</td>
    </tr>
    <tr>
        <td>Data 4</td>
        <td>Data 5</td>
        <td>Data 6</td>
    </tr>
</table>

Output

This code will produce the following table:

Header 1 Header 2 Header 3
Data 1 Data 2 Data 3
Data 4 Data 5 Data 6

The table is structured as follows:

  • <table>: Defines the table.
  • <tr>: Defines a row.
  • <th>: Defines a header cell.
  • <td>: Defines a data cell.

Use this structure to create your own tables in HTML!

Forms and Input

HTML Tutorial: Forms and Inputs

In this tutorial, we will learn how to create forms and inputs in HTML.

1. Basic Form Structure

To create a form, use the <form> tag. Here's an example:

<form action="submit_form.php" method="post">
              <!-- Form content goes here -->
          </form>

2. Adding Input Fields

Input fields are added using the <input> tag. Common types of input fields include text, email, password, and submit. Here's an example:

<form action="submit_form.php" method="post">
              <label for="name">Name:</label>
              <input type="text" id="name" name="name">
              <br>
              <label for="email">Email:</label>
              <input type="email" id="email" name="email">
              <br>
              <label for="password">Password:</label>
              <input type="password" id="password" name="password">
              <br>
              <input type="submit" value="Submit">
          </form>

3. Adding Labels

Labels make forms more accessible. Use the <label> tag to associate a label with an input field. Example:

<label for="username">Username:</label>
          <input type="text" id="username" name="username">

4. Example: Complete Form

Below is an example of a complete form:

<form action="submit_form.php" method="post">
              <label for="firstName">First Name:</label>
              <input type="text" id="firstName" name="firstName">
              <br>
              <label for="lastName">Last Name:</label>
              <input type="text" id="lastName" name="lastName">
              <br>
              <label for="emailAddress">Email:</label>
              <input type="email" id="emailAddress" name="emailAddress">
              <br>
              <input type="submit" value="Register">
          </form>

Multimedia in HTML

HTML Multimedia Tutorial

HTML allows you to integrate multimedia elements such as audio, video, and images to create engaging content for users.

1. Adding Audio

To add an audio file in HTML, use the <audio> tag.

<audio controls>
  <source src="example.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

2. Adding Video

To embed a video in your webpage, use the <video> tag.

<video controls width="600">
  <source src="example.mp4" type="video/mp4">
  Your browser does not support the video element.
</video>

3. Embedding Images

Images can be added using the <img> tag with the src attribute for the image path and alt text for accessibility.

<img src="example.jpg" alt="Example Image" width="600">

Multimedia elements in HTML enrich the user experience and make your website more interactive.

Practice using these tags and explore their attributes to customize your multimedia content.

Colors, Background and Size

: Colors, Background, and Size

In HTML, you can customize the colors, backgrounds, and sizes of elements easily using attributes.

1. Colors

To set a color, use the style attribute with the color property: 

        <p style="color: red;">This is red text.</p>

2. Background

To set a background color, use the style attribute with the background-color property: 

        <div style="background-color: yellow;">This has a yellow background.</div>

3. Size

To set the size of an element, use the style attribute with properties like font-size (for text size) or width and height: 

        <p style="font-size: 20px;">This text is 20px in size.</p>
        <div style="width: 100px; height: 50px; background-color: blue;"></div>

Example

        <div style="color: white; background-color: black; font-size: 16px;">
            Hello, this is an example of text with custom colors, background, and size!
        </div>

Div and Classes
<div> Tag:
The <div> tag is a block-level element in HTML used to group content together for styling, layout, or scripting purposes. It doesn't have any inherent style, but you can apply styles or functions to it using CSS or JavaScript.

Example:
<div>
   <p>This is a paragraph inside a div element.</p>
</div>

This code will create a container <div> around the paragraph, but visually, it won't make a difference unless we add styles like margins, padding, or background color. The real power of the <div> tag is in its ability to structure the page.

When to use <div>:
The <div> tag is ideal for grouping elements like text, images, or even other containers. It's commonly used in layouts, especially when you're building a web page with multiple sections.

Classes:
A class is an attribute you can assign to HTML elements to group them and apply the same styles or JavaScript functions. Unlike the <div> tag, which is just a container, classes allow you to identify and manipulate multiple elements at once using CSS or JavaScript.

Example:
<div class="example-class">
   <p>This is a paragraph inside a div with a class.</p>
</div>

In this example, the <div> has the class "example-class". This class can be used to target the element with CSS or JavaScript. For instance, if you want to change the background color of all elements with the "example-class", you can use:
<style>
  .example-class {
    background-color: lightblue;
  }
</style>

Multiple Classes:
You can assign multiple classes to an element by separating them with a space. This allows you to combine different styles or behaviors. For example:
<div class="class-one class-two">
   <p>This div has two classes: "class-one" and "class-two".</p>
</div>

Why Use Classes?
Classes allow you to apply common styling or behaviors to multiple elements without affecting other elements on the page. You can have different classes for colors, fonts, layouts, or animations. It's an efficient way to manage styles across large projects.

More Examples:
Here's how classes can be used for styling purposes:
<div class="container">
   <div class="header">Header Content</div>
   <div class="content">Main Content</div>
   <div class="footer">Footer Content</div>
</div>

In this example, "container", "header", "content", and "footer" are different classes that can be styled separately to create a webpage layout.

Now you understand how to use the <div> tag for grouping content and how to use classes to apply styling or behavior to elements.
