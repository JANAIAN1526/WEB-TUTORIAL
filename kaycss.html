Introduction to CSS

Introduction to CSS

CSS, which stands for Cascading Style Sheets, is a powerful tool used to control the layout and appearance of web pages. It allows you to apply styles such as colors, fonts, and spacing to HTML elements, making websites more visually appealing and easier to navigate.

By using CSS, you can control the presentation of your content. It separates the structure (HTML) from the styling (CSS), making web development more organized and efficient.

CSS can be added to HTML in three ways:

  • Inline: Using the style attribute directly in an HTML element.
  • Internal: By placing CSS inside a <style> tag in the <head> section.
  • External: By linking to an external stylesheet using the <link> tag.

CSS Code Examples

Here are some examples of how to add CSS to an HTML document:

Inline CSS

<div style="color: blue; background-color: yellow;">This is an inline styled div.</div>

Internal CSS

<style>
          body {
            background-color: #282c34;
            color: white;
          }
          h1 {
            text-align: center;
          }
        </style>

External CSS

<link rel="stylesheet" href="styles.css">

In the examples above, you can see how different methods are used to apply CSS styles to HTML elements.

CSS Colors and Fonts

Colors and Fonts

CSS Tutorial: Colors and Fonts

This tutorial covers:

  • How to use colors in CSS (hex, rgb, rgba, hsl).
  • Text styling properties like alignment, decoration, and transformation.
  • Font customization including size, style, and weight.

1. Colors in CSS

CSS provides several ways to specify colors:

              
      /* Hexadecimal color example */
      p {
          color: #ff5733; /* Sets text color to a bright orange */
      }
      
      /* RGB color example */
      p {
          color: rgb(255, 87, 51); /* Same as #ff5733 in RGB format */
      }
      
      /* RGBA color example */
      p {
          color: rgba(255, 87, 51, 0.5); /* Same as above, but with 50% transparency */
      }
      
      /* HSL color example */
      p {
          color: hsl(14, 100%, 60%); /* Defines color using hue, saturation, and lightness */
      }

2. Text Styling

Common text styling properties in CSS:

              
      p {
          text-align: center; /* Aligns text to the center */
          text-decoration: underline; /* Adds an underline to the text */
          text-transform: uppercase; /* Converts text to uppercase */
      }

3. Font Customization

CSS allows you to control font appearance with the following properties:

              
      p {
          font-size: 20px; /* Sets the size of the font */
          font-weight: bold; /* Makes the font bold */
          font-style: italic; /* Italicizes the font */
          font-family: 'Times New Roman', serif; /* Uses Times New Roman as the primary font */
      }

Explanation of Code

Colors:

  • Hexadecimal colors (e.g., #ff5733) are defined using a six-digit code.
  • RGB (e.g., rgb(255, 87, 51)) defines colors using red, green, and blue values.
  • RGBA (e.g., rgba(255, 87, 51, 0.5)) adds an alpha channel for transparency.
  • HSL (e.g., hsl(14, 100%, 60%)) uses hue, saturation, and lightness for color definition.
Text Styling:
  • text-align: Aligns text (e.g., left, center, right).
  • text-decoration: Adds effects like underline, line-through, or overline.
  • text-transform: Converts text to uppercase, lowercase, or capitalized.
Font Customization:
  • font-size: Specifies the size of text (e.g., 20px, 1.5em).
  • font-weight: Controls the thickness of text (e.g., normal, bold, 700).
  • font-style: Sets the text style (e.g., normal, italic, oblique).
  • font-family: Defines the font type (e.g., Arial, serif).

CSS Box

CSS Tutorial: The Box Model

The CSS Box Model is a fundamental concept in web design. It describes how elements are structured and spaced on a webpage.

Structure of the Box Model

The Box Model consists of:

  • Content: The actual content inside the box, such as text or an image.
  • Padding: The space between the content and the border.
  • Border: The line surrounding the padding (or content if there's no padding).
  • Margin: The space outside the border, separating the element from other elements.

Box Model in Action

Hereâ€™s an example of how the Box Model is defined in CSS:

        
/* Example CSS for Box Model */
div {
    width: 200px; /* The width of the content area */
    height: 100px; /* The height of the content area */
    padding: 20px; /* Space between content and border */
    border: 5px solid lightblue; /* A 5px wide border */
    margin: 10px; /* Space outside the border */
}

Explanation of Code

  • Content: The width and height properties define the size of the content area.
  • Padding: Adds 20px of space between the content and the border. This increases the total size of the element.
  • Border: A 5px solid border in light blue surrounds the padding and content.
  • Margin: Adds 10px of space outside the border, creating separation from other elements.

How the Box Model Affects Layout

The total size of an element is calculated as:

Total Width = Content Width + Padding + Border + Margin

Total Height = Content Height + Padding + Border + Margin

In the above example, the total width of the box would be:

        
200px (content width) + 20px (left padding) + 20px (right padding)
+ 5px (left border) + 5px (right border) + 10px (left margin) + 10px (right margin)
= 270px

Similarly, the total height would be 150px.

CSS Flexbox

Introduction to Flexbox

The CSS Flexbox (Flexible Box) layout is a powerful tool for creating flexible and responsive layouts. It allows you to align and distribute space among items in a container, even when their sizes are dynamic or unknown. Flexbox is especially useful for designing layouts that work well on various screen sizes.

How Flexbox Works

Flexbox uses the concept of a flex container and flex items. The container is the parent element, and the items are its children. When you apply display: flex; to a container, its children become flex items that can be aligned and arranged using Flexbox properties.

Example: Flexbox Layout

Here is an example of how to use Flexbox:

            
    <div style="display: flex; justify-content: space-around; align-items: center; height: 100px; background-color: white;">
        <div style="width: 50px; height: 50px; background-color: white;"></div>
        <div style="width: 50px; height: 50px; background-color: white;"></div>
        <div style="width: 50px; height: 50px; background-color: white;"></div>
    </div>

Explanation of the CSS Code

The CSS properties used in the above example are explained below:

  • display: flex; - Turns the container into a flex container.
  • justify-content: space-around; - Distributes flex items with equal space around them.
  • align-items: center; - Aligns flex items vertically at the center.
  • background-color; - Specifies the background color of elements.

Why Use Flexbox?

Flexbox makes it easy to create layouts that:

  • Adapt to different screen sizes
  • Align items horizontally and vertically with minimal effort
  • Handle dynamic content gracefully

Understanding CSS Grid

CSS Grid is a powerful layout system that allows you to create flexible and complex layouts with rows and columns. It provides a two-dimensional grid-based layout system, which makes it easier to design web pages without using floats or positioning.

Below, we will demonstrate how to create a grid layout with inline styles only. This includes defining rows and columns, and positioning elements using grid properties like grid-column and grid-row.

How This Works

The grid container is created with the display: grid property. Here are the key features of this layout:

display: flex; - Turns the container into a flex container.> display: flex; - Grid Columns: The property grid-template-columns: repeat(4, 1fr) defines four equal columns. display: flex; - The 1fr means each column takes an equal fraction of the available space.
  • display: flex; - Grid Rows: The rows adjust automatically to fit their content because we used grid-template-rows: auto.
  • display: flex; - Spacing: The gap property adds spacing between grid items (in this case, 10px).
  • display: flex; - Spanning: The grid-column: span 2 makes an item span two columns, display: flex; - and grid-row: span 2 makes an item span two rows.

    • Positioning Elements in CSS Grid

    CSS Grid allows precise positioning of elements within the grid. For example:

    • Use grid-column-start and grid-column-end to define the start and end positions of an item within the columns.
    • Use grid-row-start and grid-row-end for rows.

    CSS Code Example

    Below is the CSS code for the grid layout used in the example:

        /* CSS Grid Layout */
    .grid-container {
        display: grid;
        grid-template-columns: repeat(4, 1fr);
        grid-template-rows: auto;
        gap: 10px;
        padding: 20px;
        background-color: white;
        border: 1px solid #ddd;
    }

        padding: 20px;
        text-align: center;
        color: black;
    }
    
    /* Spanning Items */
    .grid-item-span-2-columns {
        grid-column: span 2;
    }
    
    .grid-item-span-2-rows {
        grid-row: span 2;
    }

    CSS Image and Button

    Learn how to add a background image and style buttons with CSS.

    CSS Code:

            <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>CSS Tutorial: Background Image & Button</title>
            <style>
                /* 
                The body of the page is given a background image and text color is set to white. 
                The background image covers the entire viewport.
                */
                body {
                    background-image: url('background.jpg'); /* Add your image path here */
                    background-size: cover; /* Make sure the image covers the whole screen */
                    color: white; /* Set all text to white */
                    font-family: Arial, sans-serif; /* Set the font to Arial */
                    margin: 0; /* Remove default margin */
                    padding: 0; /* Remove default padding */
                    height: 100vh; /* Ensure the body takes the full height of the viewport */
                    display: flex; /* Use flexbox for centering content */
                    justify-content: center; /* Center content horizontally */
                    align-items: center; /* Center content vertically */
                    text-align: center; /* Center text alignment */
                }
        
                /* 
                Styling for the button:
                - Background color is set to green.
                - Text color is white.
                - Padding and font size are set for better appearance.
                - On hover, the background color of the button changes to a darker green.
                */
                .btn {
                    background-color: #4CAF50; /* Green button */
                    color: white; /* Text color */
                    padding: 15px 32px; /* Add padding to the button */
                    text-align: center; /* Center the text in the button */
                    text-decoration: none; /* Remove underline from text */
                    display: inline-block; /* Button is an inline-block element */
                    font-size: 16px; /* Set the font size */
                    margin: 4px 2px; /* Margin for spacing between buttons */
                    cursor: pointer; /* Change the cursor to a pointer when hovering over the button */
                    border: none; /* Remove default border */
                    border-radius: 4px; /* Rounded corners */
                    transition: background-color 0.3s ease; /* Smooth transition for hover effect */
                }
        
                /* 
                The hover effect changes the button background to a darker shade of green when the mouse hovers over it.
                */
                .btn:hover {
                    background-color: #45a049; /* Darken on hover */
                }
            </style>
        </head>
        <body>
            <div>
                <h1>Welcome to the CSS Tutorial!</h1>
                <p>Learn how to add a background image and style buttons with CSS.</p>
                <button class="btn">Click Me!</button>
            </div>

                The 'background-size: cover' ensures the image covers the entire screen area.</li>

     We set the text color to white using 'color: white;' so that all text in the page appears white.</li>
