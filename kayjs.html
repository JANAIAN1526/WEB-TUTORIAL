Introduction to JS

Introduction to JavaScript

What is JavaScript?
JavaScript is a programming language that adds interactivity to web pages. It allows you to create dynamic, responsive, and engaging user experiences.

Why is JavaScript Important?
JavaScript is essential for modern web development. It's used to:

  • Create interactive elements like buttons, sliders, and menus
  • Handle user input and events
  • Animate elements on the page
  • Fetch and display data from servers
  • Build complex web applications

JavaScript vs. HTML vs. CSS


While HTML, CSS, and JavaScript are often used together, they have distinct roles:

  • HTML defines the structureof a webpage.
  • CSS styles the appearance of a webpage.
  • JavaScript adds interactivity and behavior to a webpage.

Modern JavaScript Applications


JavaScript has evolved significantly, powering a wide range of applications:

  • Web Applications:Single-page applications (SPAs) and multi-page applications (MPAs)
  • Game Development:Creating interactive games for web and mobile platforms
  • Server-Side Development:Node.js for building server-side applications
  • Mobile App Development:Frameworks like React Native and Ionic for building cross-platform apps
  • Desktop App Development: Electron for building desktop applications using web technologies

Setting up Environment

Setting Up the Development Environment

1. Installing a Text Editor (e.g., VS Code)

A text editor is essential for writing and editing your code. Visual Studio Code (VS Code) is a popular choice among developers due to its user-friendly interface, built-in terminal, and extensive extensions. To install it:

  1. Visit the VS Code website.
  2. Download the installer for your operating system (Windows, macOS, or Linux).
  3. Run the installer and follow the on-screen instructions to complete the installation.

2. Setting Up a Basic HTML File to Include JavaScript

Once you have installed your text editor, create a new file with the extension .html. This file will contain your HTML code, and you can also include JavaScript within it. Here is a simple example of what your HTML file might look like: 

            
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>My First HTML Page</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
                <script>
                    console.log('Hello from JavaScript!');
                </script>
            </body>
            </html>

3. Introduction to Browser Developer Tools

Developer tools are built into modern web browsers and allow you to inspect and debug your HTML, CSS, and JavaScript. To access these tools, right-click on any webpage and select Inspect or press F12 on your keyboard. Here are some key features:

  • Elements Tab: View and edit the HTML and CSS of a webpage in real-time.
  • Console Tab: View messages logged by JavaScript and run JavaScript commands directly.
  • Network Tab: Monitor network requests and responses, which is useful for debugging API calls.

JS Syntax and Basics

JavaScript Syntax and Basics

1. Variables

In JavaScript, variables are used to store data values. You can declare variables using three keywords: var, let, and const.

  • var: This keyword declares a variable that is function-scoped or globally-scoped. It can be re-assigned and re-declared. 
    var name = "Alice";
  • let: This keyword declares a block-scoped variable. It can be re-assigned but not re-declared in the same scope. 
    let age = 30;
  • const: This keyword declares a block-scoped variable that cannot be re-assigned. It must be initialized at the time of declaration. 
    const pi = 3.14;

2. Data Types

JavaScript has several data types that can be categorized into two groups: Primitive and Reference types.

Primitive Data Types:

  • String: Represents a sequence of characters. 
    let greeting = "Hello, World!";
  • Number: Represents both integer and floating-point numbers. 
    let count = 42;
  • Boolean: Represents a logical entity and can have two values: true or false. 
    let isActive = true;
  • null: Represents an intentional absence of any value. 
    let emptyValue = null;
  • undefined: Indicates a variable that has been declared but has not yet been assigned a value. 
    let notAssigned;

3. Comments and Semicolons

Comments are used to explain code and are ignored by the JavaScript engine. There are two types of comments:

  • Single-line comments: Start with two forward slashes (//). 
    // This is a single-line comment
  • Multi-line comments: Enclosed between /* and */. 
    /* This is a multi-line comment */

In JavaScript, it is a good practice to end statements with a semicolon (;) to avoid potential issues, although it is not strictly required in all cases.

JavaScript operators

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations on numerical values.

Operator Description Example
+ Addition let sum = 5 + 3;
- Subtraction let difference = 10 - 4;
* Multiplication let product = 2 * 6;
/ Division let quotient = 15 / 3;
% Modulus (Remainder) let remainder = 17 % 5;

Comparison Operators

Comparison operators are used to compare two values and return a boolean value (true or false).

Operator Description Example
== Equal to let isEqual = 5 == 5;
!= Not equal to let isNotEqual = 3 != 7;
> Greater than let isGreaterThan = 10 > 8;
< Less than let isLessThan = 2 < 9;
>= Greater than or equal to let isGreaterOrEqual = 6 >= 6;
<= Less than or equal to let isLessOrEqual = 4 <= 4;

Logical Operators

Logical operators are used to combine multiple conditions and return a boolean value.

Operator Description Example
&& AND (both conditions must be true) let isTrue = (5 > 3) && (2 < 4);
|| OR (at least one condition must be true) let isTrue = (10 == 10) || (7 > 12);
! NOT (reverses the logical state) let isFalse = !(true);

Assignment Operators

Assignment operators are used to assign values to variables.

Operator Description Example
= Assigns   a value let x = 10;
+= Adds a value and assigns x += 5; // x = 15
-= Subtracts a value and assigns x -= 2; // x = 13
*= Multiplies a value and assigns x *= 3; // x = 39
/= Divides a value and assigns x /= 3; // x = 13
%= Calculates the modulus and assigns x %= 4; // x = 1

Combining Operators in Expressions

You can combine multiple operators in a single expression to create complex calculations. Remember to consider the order of operations.

Example:

let result = (10 + 5) * 2 - 3; // Output: 27

Control Structures

Conditional Statements

Conditional statements allow you to execute different code blocks based on certain conditions.

if Statement

The `if` statement executes a code block if a specified condition is true.

if (condition) { // Code to be executed if the condition is true }

if-else Statement

The `if-else` statement executes one code block if a condition is true, and another code block if the condition is false.

if (condition) { // Code to be executed if the condition is true } else { // Code to be executed if the condition is false }

if-else if-else Statement

The `if-else if-else` statement allows you to check multiple conditions and execute different code blocks based on the first true condition.

if (condition1) { // Code to be executed if condition1 is true } else if (condition2) { // Code to be executed if condition1 is false and condition2 is true } else { // Code to be executed if both conditions   are false }  

switch Statement

The `switch` statement allows you to execute different code blocks based on the value of an expression.

switch (expression) { case value1: // Code to be executed if expression equals value1 break; case value2: // Code to be executed if expression equals value2 break;   // ... more cases default: // Code to be executed if   no case matches }

Loops

Loops allow you to execute a block of code repeatedly.

for Loop

The `for` loop is used to execute a block of code a specific number of times.

for (initialization; condition; increment/decrement) { // Code to be executed }

while Loop

The `while` loop executes a block of code as long as a specified condition is true.

while (condition) { // Code to be executed }

do-while Loop

The `do-while` loop executes a block of code at least once, and then continues to execute as long as a specified condition is true.

do { // Code to be executed } while (condition);

Breaking and Continuing Loops

You can use the `break` and `continue` statements to control the flow of loops.

break Statement

The `break` statement immediately terminates the loop.

for (let i = 0; i < 10; i++) { if (i === 5) { break; // Exit the loop when i reaches 5 } console.log(i); }

continue Statement

The `continue` statement skips the current iteration of the loop and moves to the next iteration.

for (let i = 0; i < 10; i++) { if (i % 2 === 0) { continue; // Skip even numbers } console.log(i); }

JS functions

Functions

Functions are blocks of reusable code that perform specific tasks.

Defining a Function

To define a function, you use the `function` keyword followed by the function name, parentheses for parameters, and curly braces for the function body.

function functionName(parameter1, parameter2, ...) { // Function body }

Calling a Function

To call a function, you simply write the function name followed by parentheses, passing any necessary arguments.

functionName(argument1, argument2, ...);

Function Parameters and Return Values

Functions can take parameters, which are values passed to the function when it's called. They can also return values using the `return` statement.

function greet(name) { console.log("Hello, " + name + "!"); } function add(x, y) { return x + y; } greet("Alice"); let sum = add(5, 3); console.log("Sum:", sum);

Arrow Functions

Arrow functions provide a concise syntax for defining functions, especially for short functions.

const square = (x) => { return x * x; }; // Concise syntax for single-line functions const double = x => x * 2;

Arrow functions are often used with higher-order functions like `map`, `filter`, and `reduce`.

const numbers = [1, 2, 3, 4, 5]; const doubledNumbers = numbers.map(number => number * 2); console.log(doubledNumbers);

Object and Arrays

Objects

Objects are used to store collections of key-value pairs.

Creating an Object

let person = { firstName: "John", lastName: "Doe", age: 30, city: "New York" };

Accessing Object Properties

console.log(person.firstName); // Output: John console.log(person["lastName"]); // Output: Doe

Adding and Removing Properties

person.email = "johndoe@example.com"; // Add a property delete person.city; // Remove a property

Arrays

Arrays are used to store collections of values.

Creating an Array

let numbers = [1, 2, 3, 4, 5]; let fruits = ["apple", "banana", "orange"];

Accessing Array Elements

console.log(numbers[0]); // Output: 1 console.log(fruits[2]); // Output: orange

Array Methods

Arrays provide various methods for manipulating their elements.

  • push(): Adds elements to the end of an array.
  • pop():Removes the last element from an array.
  • map(): Creates a new array by transforming each element of the original array.
  • filter(): Creates a new array with elements that meet a certain condition.
numbers.push(6); // Add 6 to the end numbers.pop(); // Remove the last element let doubledNumbers = numbers.map(number => number * 2); let evenNumbers = numbers.filter(number => number % 2 === 0);

Object Methods and Iteration

Objects can have methods, which are functions associated with the object.

let person = { firstName: "John", lastName: "Doe", fullName: function() { return this.firstName + " " + this.lastName; } }; console.log(person.fullName());  

You can iterate over object properties using a `for...in` loop.

for (let property in person) { console.log(property + ": " + person[property]); }
